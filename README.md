# poll-config
dummy project to get scheduling running

## Steps
Command to create project

```
mvn archetype:generate -DgroupId=com.cognizant.polling -DartifactId=poll-config 
   -Dversion=1.0 -DarchetypeArtifactId=maven-archetype-webapp
```
